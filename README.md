### M5StickCで3Dオブジェクトを表示できるライブラリ  
- 複数のウィンドウ
- ワイヤーフレーム
- ポリゴン表示（Zソート法　Zバッファだけどポリゴン単位なので）
- 法線カラー表示
- Directional Light
- Ambient
- 8000ポリゴンくらいまで
